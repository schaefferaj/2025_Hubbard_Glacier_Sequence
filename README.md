# 2025_Hubbard_Glacier_Sequence
Repository containing AI-based Event Catalog (with picks and station information) for the 2025 Hubbard Glacier (St Elias Mountains, Kluane National Park, Yukon) Sequence.

This dataset accompanies the Journal Article:
C. Brillon, J.M. Gosselin, R.M.H. Dokht, A.J. Schaeffer. The December 6, 2025 Mw 7.0 earthquake in Yukon, Canada: Assessing routine event data in a remote northern setting. Seismic Fast Reports, submitted April 5th, 2026.
